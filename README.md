# Welcome to Money Cents

## Overview

MoneyCents is a dynamic website developed in Visual Studio Code using Bootstrap-React, React, CSS, Node.js for front-end styling, Express for session management, PostgreSQL for user databases, and bcrypt for password hashing. The application is hosted on GitHub Code Spaces, ensuring a seamless and collaborative development environment. 

Money Cents is a comprehensive resource designed for women at every life stage, aiming to demystify and eliminate the stigma surrounding financial matters. The platform offers curated financial tips and tricks sourced from around the World Wide Web, covering topics such as saving, investing, community insights, and financial planning providing accessible information in a easily accessible Space 

## Features

- **Responsive Design:** Utilizing Bootstrap-React and CSS, FinanceHub ensures an optimal viewing experience across various devices.

- **User Authentication:** Securely manage user sessions with Express and enhance password security through bcrypt hashing.

- **Database Integration:** PostgreSQL is employed to efficiently handle user databases, ensuring reliable storage and retrieval of information.

- **Finance Blogs:** Explore a curated collection of finance-related blogs on the dedicated page, offering valuable insights and resources.
